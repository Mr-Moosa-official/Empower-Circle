# **App Name**: EmpowerCircle

## Core Features:

- Home Dashboard: Home dashboard with quick access cards for SOS, location tracking, HelpNet and SkillHub.
- SOS Emergency Button: A prominent SOS button with location sharing and fake call functionality.
- HelpNet Support Page: HelpNet page displaying profiles of women needing support with donation options.
- SkillHub Learning Resources: SkillHub learning page with videos and resources for self-defense, rights, and financial skills.
- Anonymous Reporting: Anonymous reporting form with voice note upload for secure incident reporting.

## Style Guidelines:

- Primary color: Lavender for calmness and safety.
- Secondary color: Teal for trust and support.
- Neutral color: White for clarity and simplicity.
- Accent: Deep pink for empowerment and action.
- Empowering fonts for readability and a sense of strength.
- Empowered female avatar as the user profile icon.
- Clean, intuitive navigation with a prominent SOS button.
- Subtle animations for the SOS feature and donation confirmation.
